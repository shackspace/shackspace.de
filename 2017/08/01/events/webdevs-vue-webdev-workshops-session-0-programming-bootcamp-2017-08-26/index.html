Webdevs: Vue Webdev Workshops: Session 0 – Programming Bootcamp

Am Wochenende vom 26. - 27. August 2017 ab 10:30 Uhr starten die Vue Webdev Workshops mit der ersten Session – dem Programming Bootcamp.

Die Workshopreihe wird euch moderne Web-Application-Entwicklung mit JavaScript/HTML/CSS und Vue beibringen. Die ersten Sessions dienen dem Wissenausgleich in den Grundlagentechnologien, um Allen die Möglichkeit zum Einstieg zu geben.

Genau zu diesem Ziel fangen wir bei 0 an – Programmieren lernen. Wer also noch nicht coden kann und gerne mit Webentwicklung anfangen möchte, ist bei dieser ersten Session genau richtig.

Eine Übersicht über die späteren Sessions gibt es hier:

https://pads.shackspace.de/webvueworkshops

Wer die Grundlagen schon beherrscht, kann problemlos in späteren Sessions einsteigen.

Um ein besseres Lernerlebnis zu schaffen, sind die Einsteigerworkshops auf fünf Plätze begrenzt. Meldet euch also hier an:

https://doodle.com/poll/yf2r6piguit34wfy

Wenn ihr an dem Wochenende keine Zeit habt, aber trotzdem Interesse habt, meldet euch im Negativ an, dann gibt es evtl. einen zweiten Termin.
