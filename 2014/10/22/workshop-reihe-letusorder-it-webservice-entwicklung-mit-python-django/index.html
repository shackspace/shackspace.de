Workshop Reihe: LetUsOrder.it - Webservice Entwicklung mit Python & Django

Am Freitag, den 31. Oktober 2014 um 19 Uhr trifft sich im shackspace die Stuttgart-Python-Interest-Group (Stupig, Protokolle der vergangenen Treffen) und startet ein Django-Einsteigerprojekt. Falls Ihr also schon immer einen Anlass gesucht hattet euch in Python und Django einzuarbeiten, hier ist die Gelegenheit.

Umgesetzt wird das Projekt LetUsOrder.It, eine Platform die das gemeinsame Bestellen von $Dingen vereinfachen soll.

Im Laufe dieses Projektes soll gemeinsam ein Webservice entwickelt werden.
Entwickelt wird ausschließlich in der Gruppe zu den zweiwöchig stattfindenden Freitagsterminen der Stupig. Das heisst, alle Änderungen am Code (auch Bugfixes) werden nur an Workshop-Terminen gemacht und niemand bleibt auf der Strecke.

Der vorläufige “Projektplan” findet sich bereits im Projektrepo auf GitHub.

  • Jeder Python/Django-Interessierte ist eingeladen mitzumachen.
  • Die Teilnahme ist kostenlos.
  • Bitte unbedingt einen internetfähigen Laptop mit funktionierendem Python 3 mitbringen.
    Wir freuen uns auf euren Besuch!
